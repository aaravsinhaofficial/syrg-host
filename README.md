# SYRG Labs

Static GitHub Pages host for the Sinha-Yalageri Research Group site.

The site is intentionally no-build: GitHub Pages can serve `index.html`,
`styles.css`, `data.js`, and the checked-in `logos/` assets directly from the
`main` branch root.

Local preview:

```sh
python3 -m http.server 4173
```

Then open `http://127.0.0.1:4173/`.
